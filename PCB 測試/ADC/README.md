# ADC 測試  
進行此測試需將`U1001 BL DEMUX`和`U1016 BL DEMUX`關掉。  
  
`U1006 MUX`的`EN`訊號給`1.8 V`，連接`PBL1`。  
  
觀測`CN1007`針腳的轉換結果。
