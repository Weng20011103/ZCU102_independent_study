# INA: INA310A5IDGKR  
# 腳位  
|Pin|符號|說明|
|:--:|:--:|:---|
|1|VS|接 + 3.3 V，接上 0.1 μF 的去耦電容|
|2|OUT|電流轉電壓輸出|
|3|CMPIN|比較器輸入|
|4|GND|接地|
|5|RESET|比較器控制器|
|6|CMPOUT|比較器輸出|
|7|IN-|連接電阻 (靠近地那端)|
|8|IN+|連接電阻 (靠近電流鏡輸出那端)|
